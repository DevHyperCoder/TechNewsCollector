# TechNewsCollector
This is a Python Script that scrapes news sites for information about new leaked products and other tech stuff

This Python script uses ```BeautifulSoup``` to scrape different websites and shows them in a GUI using ```Tkinter``` This project is just for personal use, NOT FOR COMMERCIAL USE

## How to run?
1. Clone this repository using ```git clone https://github.com/DevHyperCoder/TechNewsCollector/```
2. Install all the required packages using ```pip install -r requirements.txt``` (Make sure you run this after you activate your virtual environment)
3. Run the main file ```python main.py```

##Required Packages
```requests```
```beautifulsoup```
